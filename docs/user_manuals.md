# Role-Based User Manuals

This manual outlines daily operating tasks and workflows for hospital staff at CHC Bharno.

---

## 1. Receptionist Workflow
- **Patient Intake**: Register new patients under the **Patient Registration** form. Issue a UHID card.
- **Appointment Booking**: Select a doctor, department, target slot, symptoms, and click **Book OPD Slot**.
- **Check In**: Upon patient arrival, check them in from the slots directory to send them to the doctor's queue.

---

## 2. Doctor Workflow
- **Consultation Workspace**: Access active patient queue lists. Click on a patient to view their profile.
- **Clinical Entry**: Record symptoms, diagnoses, prescribe medicines (linked to pharmacy stock), and raise laboratory orders.
- **Leave Request**: File time off from the Roster section. Appointments will automatically be flagged for rescheduling.

---

## 3. Laboratory Technician Workflow
- **Test Queues**: View laboratory orders raised by doctors.
- **Result Recording**: Perform analyzers, input parameters (e.g. haemoglobin levels), and click **Submit Lab Report** to update the patient clinical dashboard instantly.

---

## 4. Pharmacist Workflow
- **Dispense Queue**: View completed doctor consultations with active prescriptions.
- **Inventory Check**: Scan stock availability, check dosage instructions, select items, and process dispensation (which automatically updates inventory counts).

---

## 5. Patient Portal
- **Dashboard Overview**: Check historical medical reports, laboratory results, active prescriptions, and self-schedule/reschedule upcoming OPD appointments.
